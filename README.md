**Banking Management System**

# Overview

A Java-based Banking Management System developed using JDBC and MySQL to manage customer accounts, loans, and banking operations efficiently.

# Features

* Customer Management
* Account Creation and Management
* Loan Processing and partial loan payments
* Generate account statements
* Database Connectivity using JDBC
* MySQL Integration for persistent data storage
* Menu-driven Banking Operations
* Credit Score Evaluation for loan processing
